# 🎂 Aniversário de Brasília

Site institucional desenvolvido como trabalho da disciplina de **Front-End**, com o tema **"Aniversário de Brasília"**. O projeto explora a história, a cultura e os pontos turísticos da capital federal por meio de uma página web estática, desenvolvida individualmente com HTML e CSS.

---

## 🗂️ Estrutura do Repositório

- **Aniversario-de-Brasilia/**
  - `index.html` — Página inicial com introdução sobre Brasília
  - `anivesário.html` — Página sobre a história e o aniversário da cidade
  - `anivesário.css` — Estilização da página do aniversário
  - `cultura.html` — Página sobre a cultura brasiliense
  - `cultura.css` — Estilização da página de cultura
  - `galeria.html` — Página com galeria de imagens da cidade
  - `galeria.css` — Estilização da galeria de imagens
  - `styles.css` — Folha de estilos principal com tema geral do site
  - `img/` — Pasta com imagens de pontos turísticos e símbolos de Brasília
    - `Brasília1.jpeg`, `Congresso.jpg`, `Eu_Amo_Brasilia.jpg`, `brasília2.jpg`, `brasília3.webp`, `brasília4.jpg`, entre outras

---

## 🛠️ Tecnologias

- **Linguagens:** HTML5, CSS3
- **Design:** CSS puro, layout responsivo, paleta de cores temática
- **Navegadores:** Google Chrome, Mozilla Firefox, Microsoft Edge

---

## 📖 Conteúdo do Site

| Página | Descrição |
|--------|-----------|
| **index.html** | Página inicial com apresentação do tema e navegação principal |
| **anivesário.html** | Histórico da fundação de Brasília, contexto do aniversário da cidade e curiosidades |
| **cultura.html** | Aspectos culturais da capital: gastronomia, música, arquitetura e tradições locais |
| **galeria.html** | Galeria de imagens com pontos turísticos icônicos (Congresso Nacional, Catedral, Ponte JK, etc.) |

---

## 🚀 Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador web moderno:

```bash
# Linux
firefox index.html

# Windows
start index.html
```

---

## 💡 Destaques do Projeto

- **Tema Histórico-Cultural:** Conteúdo pesquisado e desenvolvido para celebrar o aniversário de Brasília, abordando aspectos históricos, culturais e arquitetônicos da cidade
- **Múltiplas Páginas:** Site estruturado em 4 páginas interligadas, cada uma com seu próprio arquivo CSS para estilização específica
- **Galeria Visual:** Seleção de imagens representativas dos principais pontos turísticos e símbolos da capital federal
- **Projeto Individual:** Desenvolvimento completo por um único aluno, abrangendo desde a pesquisa de conteúdo até a implementação front-end
